BLACK ADMIN - COMO USAR

1. Abra admin/index.html no navegador.
2. Entre com a senha padrão: BlackInLove@2026
3. Edite os links/datas desejados.
4. Clique em "Baixar config.js".
5. Substitua o arquivo js/config.js do projeto pelo arquivo baixado.
6. Suba a alteração no GitHub.

IMPORTANTE:
- Este painel é estático, feito para GitHub Pages.
- Ele NÃO salva direto no GitHub sozinho, porque GitHub Pages não possui backend.
- A senha é uma barreira visual/local, não segurança real de servidor.
- Para segurança exclusiva real, o painel administrativo deve ficar em backend separado com login e banco de dados.
