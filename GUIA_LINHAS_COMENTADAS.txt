PROJETO FLAYER - VERSAO COM LINHAS COMENTADAS

O arquivo principal comentado e:
  css/style.css

Tambem deixei uma copia de estudo em:
  css/style_COMENTADO_LINHA_A_LINHA.css

COMO LER NO VS CODE:
- Tudo que esta depois de /* ... */ e comentario.
- O comentario nao aparece no site e nao altera o funcionamento.
- Para mover uma caixa: procure left / top.
- Para alterar tamanho: procure width / height / font-size.
- Para mudar brilho/sombra: procure filter / box-shadow / text-shadow.
- Para mudar camada: procure z-index.

EXEMPLO:
  left: 2.95% !important; /* Posição horizontal. Menor vai para esquerda, maior vai para direita. */

IMPORTANTE:
- Mantive o projeto funcional.
- O foco dos comentarios linha a linha foi o CSS, porque e onde voce mais mexe visualmente.
- O HTML e o JS ja tinham comentarios explicativos dos blocos principais.
