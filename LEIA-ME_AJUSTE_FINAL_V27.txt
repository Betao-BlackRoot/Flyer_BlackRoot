PROJETO FLAYER - AJUSTE FINAL V27

O QUE FOI AJUSTADO

1) Seu HTML agora pode ficar só com o flyer.
   Você removeu o cabeçalho, o player lateral e o botão flutuante.
   Antes isso quebrava o app.js, porque o JavaScript ainda procurava esses elementos.
   Agora o app.js foi protegido para funcionar mesmo quando esses blocos não existem.

2) O selo AO VIVO ficou 100% dinâmico.
   Estados atuais:
   - PAUSADO: nada tocando ou botão Pausar acionado.
   - VINHETA: chamada.mp3 tocando.
   - CONECTANDO: tentando conectar na rádio.
   - AO VIVO: só quando o navegador confirma que a rádio realmente começou a tocar.
   - ERRO: falha ao tocar vinheta ou rádio.

3) Foi adicionado o botão PAUSAR no menu do flyer.
   Agora, como o player externo foi removido, você ainda consegue pausar pelo próprio flyer.
   Caminho no HTML:
   id="choicePauseBtn"

4) O flyer foi centralizado.
   Como não existe mais player lateral, o CSS final centraliza a experiência na tela.
   Procure no CSS por:
   AJUSTE FINAL V27 - FLYER SOLO + JS ROBUSTO

ONDE MEXER NO VS CODE

- Para trocar textos do selo:
  js/app.js
  procure por: atualizarSeloAoVivo

- Para trocar a rádio:
  js/app.js
  procure por: const STREAM

- Para trocar links oficiais:
  js/app.js
  procure por: const LINKS

- Para mexer na posição/tamanho do flyer solo:
  css/style.css
  procure por: AJUSTE FINAL V27

- Para mexer no botão Pausar:
  index.html
  procure por: choicePauseBtn
